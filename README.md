<h1>Classificador de Heróis</h1>

Um programa em JavaScript que classifica heróis com base na experiência (XP), exibindo o nível correspondente diretamente no console. O sistema permite testar múltiplos heróis, calculando e mostrando seu status de forma organizada.

- **Funcionalidades:**
  - **Cadastro de heróis:** Armazena o nome e a quantidade de XP de cada personagem.
  - **Classificação por níveis:** Determina o nível do herói (Ferro, Bronze, Prata, Ouro, Platina, Ascendente, Imortal ou Radiante) com base na XP.
  - **Estruturas de decisão:** Utiliza condicionais para definir o nível correto de cada herói.
  - **Testes pré-definidos:** Permite testar múltiplos heróis com valores de XP já definidos no código.
  - **Saída clara:** Exibe no console o nome do herói junto com seu nível de forma organizada.
